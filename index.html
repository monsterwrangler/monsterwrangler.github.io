Custom Icons for BostonPogoMap

BostonPogoMap.com is a great Pokémon Go map, but sadly, they were asked to remove the icons for the Pokémon. Fortunately you can get them back using a "bookmarklet" -- a bookmark that contains JavaScript to fix up the icons. The process is a little odd, so we'll go through it step by step.

To get started, go to this bookmarklet page (the rest of the instructions are on that page).
